我发现了，blender入门教程其实就是教快捷键。

## 快捷键

|                             功能                             |                            快捷键                            |
| :----------------------------------------------------------: | :----------------------------------------------------------: |
|                         进入编辑模式                         |                             TAB                              |
|                         切换选择模式                         |                              W                               |
| 环选，例如点竖线会把横着一圈选中，而点横线会把竖着的一圈选中 |                           alt+左键                           |
|                             连选                             |                             ctrl                             |
|       多选（这个和上一个刚好和windows系统的操作相反）        |                            shift                             |
|                             全选                             |                              A                               |
|                           取消全选                           |                            双击A                             |
|                             挤出                             |                              E                               |
|         环切，环切确认的时候滚动滚轮可以一次切好几个         |                            ctrl+R                            |
|                             倒角                             |                            ctrl+B                            |
|                            内插面                            | I（进编辑模式，双击I会有不太一样的效果，但是我说不清是什么样的） |
|                             环切                             |                            ctrl+R                            |
|                          偏移环切线                          |                         ctrl+shift+R                         |
