# Ren'py引擎的使用

## 安装

1. Renpy可以在[官方网站](https://www.renpy.org/)下载；
2. 除了安装引擎本身以外，还需要准备python2和python3；
3. 解包游戏文件需要使用python3安装unrpa，命令行中的安装命令为`py -3 -m pip install unrpa`；
